# 🎬 Criador de Vídeos com Trailer

## ✨ Nova Funcionalidade

Agora você pode criar vídeos promocionais usando os **trailers oficiais** dos filmes e séries como fundo!

## 🎯 Como Funciona

### 1. **Busque o Filme/Série**
- Use a busca normal para encontrar o título
- Selecione o filme ou série desejado

### 2. **Clique em "Criar Vídeo com Trailer"**
- Botão vermelho abaixo do "Gerar Banner"
- Abre o modal de criação de vídeo

### 3. **Sistema Busca o Trailer Automaticamente**
- Busca primeiro em português (pt-BR)
- Se não encontrar, busca em inglês (en-US)
- Prioriza trailers oficiais

### 4. **Opções Disponíveis**

#### Duração do Vídeo
- **15 segundos** - Ideal para Stories
- **30 segundos** - Perfeito para Reels/TikTok
- **60 segundos** - Para Feed do Instagram
- **Trailer completo** - Vídeo inteiro

#### Formato
- **Stories (9:16)** - Vertical para Instagram/WhatsApp
- **Feed (4:5)** - Quadrado para feed
- **Paisagem (16:9)** - Horizontal para YouTube

#### Elementos Visuais
- ☑️ Mostrar minha logo
- ☑️ Mostrar contatos (WhatsApp/Instagram)

### 5. **Ações Disponíveis**

#### 🎬 Abrir Trailer no YouTube
- Abre o trailer em uma nova aba
- Você pode assistir antes de baixar

#### 📋 Copiar Link do Trailer
- Copia o link do YouTube
- Use para baixar com ferramentas externas

## 📱 Como Criar o Vídeo Final

### Método 1: Editor Online (Recomendado)

1. **Baixe o trailer do YouTube**
   - Use: [y2meta.is](https://y2meta.is/)
   - Ou: [ytdown.to](https://app.ytdown.to/pt27/)
   - Cole o link copiado

2. **Edite no CapCut Web**
   - Acesse: [capcut.com](https://capcut.com)
   - Importe o vídeo do trailer
   - Adicione sua logo (PNG transparente)
   - Adicione textos com seus contatos
   - Aplique filtros/efeitos
   - Exporte no formato desejado

### Método 2: App Mobile

1. **Baixe o trailer** (use o link copiado)

2. **Use o CapCut (App)**
   - Disponível para iOS e Android
   - Importe o vídeo
   - Adicione logo e textos
   - Exporte

3. **Ou use InShot**
   - Mais simples e rápido
   - Bom para iniciantes

### Método 3: Desktop Profissional

1. **Adobe Premiere Pro**
   - Para edições profissionais
   - Mais recursos e controle

2. **DaVinci Resolve** (Grátis)
   - Alternativa gratuita poderosa
   - Qualidade profissional

## 🎨 Dicas de Edição

### Logo
- Use PNG com fundo transparente
- Posicione no canto superior esquerdo
- Tamanho: 15-20% da largura do vídeo
- Adicione sombra para destacar

### Contatos
- Coloque na parte inferior
- Use fundo semi-transparente
- Fonte grande e legível
- Cores contrastantes

### Textos Adicionais
- Título do filme em destaque
- Ano de lançamento
- Gênero
- Classificação (⭐ nota)
- Call-to-action: "Disponível agora!"

### Efeitos
- Transições suaves
- Zoom nos momentos importantes
- Slow motion em cenas de ação
- Música de fundo (opcional)

## ⚠️ Observações Importantes

### Direitos Autorais
- Os trailers são propriedade dos estúdios
- Use apenas para fins promocionais
- Não monetize diretamente
- Dê créditos quando possível

### Qualidade
- Baixe sempre na melhor qualidade disponível
- Mínimo: 1080p (Full HD)
- Ideal: 4K (se disponível)

### Tamanho do Arquivo
- Stories: Máximo 4MB
- Reels: Máximo 4GB
- Feed: Máximo 4GB
- WhatsApp: Máximo 16MB

### Duração
- Respeite os limites de cada plataforma
- Vídeos curtos têm mais engajamento
- 15-30 segundos é o ideal

## 🎯 Exemplo de Workflow Completo

1. **No DivulgaFácil**:
   - Busque "Vingadores: Ultimato"
   - Clique em "Criar Vídeo com Trailer"
   - Copie o link do trailer

2. **Baixe o Trailer**:
   - Acesse y2meta.is ou ytdown.to
   - Cole o link
   - Baixe em 1080p

3. **Edite no CapCut**:
   - Importe o vídeo
   - Corte para 30 segundos (melhores cenas)
   - Adicione sua logo (canto superior esquerdo)
   - Adicione texto: "📱 (11) 99999-9999"
   - Adicione texto: "📸 @seuperfil"
   - Adicione texto: "Disponível agora!"
   - Aplique filtro "Cinemático"
   - Exporte em 1080p

4. **Publique**:
   - Instagram Reels
   - TikTok
   - YouTube Shorts
   - WhatsApp Status
